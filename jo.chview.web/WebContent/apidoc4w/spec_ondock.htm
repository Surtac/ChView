Forty Worlds REST Service API - ondock://

This returns information on cargo awaiting sale.

Scheme:ondock://
Authority:id@quad
Path:Population Center Name
Query Params:date in yyyy-ddd or dddddd format
API Params:none

Examples

  • ondock://2010712@071200010000/Havelock?date=1110-120 JSON XML
  • ondock://2010712@071200010000/Havelock?date=62803 JSON XML
