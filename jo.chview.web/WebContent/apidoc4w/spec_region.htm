Forty Worlds REST Service API - region://

This returns a region of space. This is a list of stars occuring in the area defined.

Scheme:region://
Authority:sphere | cube | quad
Path:x/y/z/r | x1/y1/z1/x2/y2/z2 | quadNum
Query Params:none
API Params:starAs = star | uri | sun | sys

Examples

  • region://quad/071100000000 JSON XML
  • region://sphere/27200/0/0/15 JSON XML
  • region://cube/27190/-10/-10/27210/10/10 JSON XML
  • starAs=uri&u=region://cube/27190/-10/-10/27210/10/10 JSON XML
  • starAs=sun&u=region://cube/27190/-10/-10/27210/10/10 JSON XML
  • starAs=sys&u=region://cube/27190/-10/-10/27210/10/10 JSON XML
