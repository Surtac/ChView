Forty Worlds REST Service API - star://

This returns information on a star.

Scheme:star://
Authority:sid@quad
Path:none
Query Params:none
API Params:starAs = star | uri | sun | sys

Examples

  • star://1000715@071500000000 JSON XML
  • starAs=uri&u=star://1000715@071500000000 JSON XML
  • starAs=sun&u=star://1000715@071500000000 JSON XML
  • starAs=sys&u=star://1000715@071500000000 JSON XML
